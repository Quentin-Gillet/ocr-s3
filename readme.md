# ocr-s3
Project made during semester 3 at epita

Requirements:
```
SDL2, SDL2_Image, gtk3+
See one your own to install this
```

Commands to use:
```
make
```
To build the exe file.
```
./main
```
To run the file then you can load a file and process it.
